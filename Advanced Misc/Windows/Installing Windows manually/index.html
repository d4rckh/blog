    Installing Windows manually

    1. Get a USB flash drive and download the Windows 10 ISO. This can be done by going to https://www.microsoft.com/en-us/software-download/windows10 (You can use an agent-switcher if you don't want to use their tool) and getting the ISO.
    2. Go to https://rufus.ie/ for Windows users, for Linux users use WoeUSB, from https://github.com/slacka/WoeUSB. Download either Rufus or WoeUSB.
    3. For Rufus users, select your USB drive under "Device." Then press "SELECT" and find your ISO file. Next hit "START."
    4. For WoeUSB users, navigate to the ISO image from the program then select USB device and hit "Install."
    5. After that is done, reboot your computer and boot the USB flash drive. (The boot menu can be F9, or many others so look up your device for more information).
    6. Now press Shift+F10 to bring up a terminal.
    7. Now is a good time to turn back if you don't know what you're doing! Type "diskpart" (without quotes) into the command prompt. Next I like to do a "list disk" so I can find what disk I want to install to.
    8. Type "select disk X" (replace X with your disk number found from list disk).
    9. Type "clean" THIS WILL WIPE YOUR DISK!!!
    10. Type "convert gpt" This converts your MBR drive to be a GPT drive.
    11. Type "create part efi size=500" This will make a partition that is 500MB.
    12. Type "format fs=fat32 quick" This will format your new partition to be fat32.
    13. Type "assign letter w" This assigns a drive letter to the efi partition.
    14. Type "create part primary" This is your primary partition. Windows will be installed to it. It will use the remainder of the disk.
    15. Type "format fs=ntfs quick" This makes your new partition NTFS.
    16. Type "assign letter c" In my case, this was already taken. If that happens to you, type "list volume" and see what drive has the c drive. After you find it type "select volume X" (replace X with volume number). Now type "remove letter=C" then after that, type "select disk X" and then "select part 2"
    17. Now you can type "exit" This closes diskpart.
    18. Type "cd /d D:\sources" This drive letter could be different for you. For me it was F, for example. If that is the case for you, go back into diskpart and type "list volume" and find the correct letter (not C: or W:)
    19. Type "dism /apply-image /imagefile:install.wim /index:1 /applydir:C:\" This might take a few minutes.
    20. Type "bcdboot C:\Windows /s W:" This creates necessary boot files.
    21. Type "regedit" Now we are in the registry! Even though it might look very complicated, just follow my steps and you will be fine.
    22. Click on "HKEY_LOCAL_MACHINE" and then click file >> Load Hive...
    23. In File name: type "C:\windows\system32\config" then press enter.
    24. Now select SOFTWARE then click open.
    25. Name it SOFT
    26. Go back into Load Hive and select SYSTEM then open, name it SYS.
    27. Now navigate to "HKEY_LOCAL_MACHINE\SOFT\Microsoft\Windows\CurrentVersion\Policies\System"
    28. Right click, New >> DWORD (32-bit) Value, name it "VerboseStatus" (Case sensitive). Double click it, switch it's value to 1.
    29. Double click "EnableCursorSuppression" Change the value to 0.
    30. Navigate to "HKEY_LOCAL_MACHINE\SYS\Setup"
    31. Double click "CmdLine" and change the text string to "cmd.exe"
    32. Close the Registry editor and then type "wpeutil reboot" to restart your computer.
    33. Wait for the setup to put a command prompt up then type "oobe\windeploy"
    34. Wait for it to say "Getting ready" then type "net user /add USERNAME" (Yes, you can change USERNAME to whatever you want).
    35. Type "net localgroup /add users USERNAME"
    36. Type "net localgroup /add administrators USERNAME"
    37. Now type "regedit" You thought we were done with it ;)
    38. Now navigate to "HKEY_LOCAL_MACHINE\SYSTEM\Setup"
    39. Double click "OOBEInProgress" and change the value to 0. Change "SetupType" to 0.
    40. Change "SystemSetupInProgress" to 0. Close regedit and type exit.
    41. This part will vary for different people, for me it instantly came up with "Choose privacy settings for your device" but it can be different for you. If it says "Please wait for the Windows Modules Installer" Then wait for at least 5 minutes, then force reboot your computer by holding down the power button. It seems to never leave that screen. According to enderman it "works most of the time."
    42. Choose your privacy settings then you're done! You have now successfully manually installed Windows 10!
